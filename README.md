# babel-webpack4-quickstart
Babel and Webpack 4 quickstart configuration

[![Software License](https://img.shields.io/badge/license-MIT-brightgreen.svg?style=flat)](LICENSE)

## Before start, install all Dependencies

```bash
npm install
```

## Development

```bash
npm run start
```

## Production

```bash
npm run build
```
